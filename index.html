VIDA NO CAMPUS

Escolinha para bebês

Foi construída em 1987

O pandavinhas é uma escolinha para bebês, onde o bebê tem a melhor experiência para aprender coisas básicasa da vida,como falar,andar,comunicar,saber o que é certo e o que é errado,até ele conseguir compreender essas coisas e ir para o Pandavas.



Refeitório/galpão

Foi construída em 1980

O refeitório/galpão é um lugar aonde a gente tem uma bancada para os nossos alunos sentarem, e do lado, fica uma mini sala de artes, para os nossos alunos fazerem a arte.



Biblioteca

Foi construída em 1982

A biblioteca é um lugar onde os professores levam para mostrar apresentações no computador por tela, ou para mostrar algum livro.
